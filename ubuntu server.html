Ubuntu Server

Ubuntu are o ediție de server care folosește aceleași depozite APT ca Ubuntu Desktop Edition. Diferențele dintre ele sunt absența unui mediu X Window într-o instalare implicită a ediției serverului (deși se poate instala cu ușurință, inclusiv Unity, GNOME, KDE sau Xfce) și unele modificări ale procesului de instalare. Ediția serverului folosește o interfață bazată pe caracter de ecran, pentru instalare, în locul unui proces grafic de instalare. Aceasta permite instalarea pe mașini cu o interfață serial sau „mută” fără suport grafic. Ediția serverului (precum versiunea desktop) acceptă virtualizarea hardware și poate fi rulată într-o mașină virtuală, fie în interiorul unui sistem de operare gazdă, fie într-un hipervizor, cum ar fi VMware ESXi, Oracle, Citrix XenServer, Microsoft Hyper-V, QEMU, a Mașină virtuală bazată pe kernel sau orice alt emulator sau virtualizator compatibil IBM PC. Modulul de securitate AppArmor pentru nucleul Linux este utilizat în mod implicit pe pachetele de software cheie, iar firewall-ul este extins la serviciile comune utilizate de sistemul de operare.

  • Are cerințe minime de: 112 MB, în timp ce un minim recomandat (fără desktop) este 512 MB RAM, și 1 GB RAM pentru „server live”, procesor 300 MHz și 1 GB spațiu pe disc (2,5 GB pentru toate funcțiile care urmează să fie instalate) .
  • Se execută toate arhitecturile majore - x86-64, ARM v7, ARM64, POWER8 și versiuni ulterioare și IBM System z mainframes prin LinuxONE.și suport inițial pentru RISC-V. SPARC nu mai este acceptată comercial.
  • Suportă ZFS, un sistem de fișiere cu capabilități de instantanee, de la Ubuntu 16.04 LTS.
  • Are LXD, un hipervizor pentru a gestiona containerele LXC Linux.
  • Include prima versiune de producție a DPDK pentru rețeaua de sâmburi de linie.
  • Utilizează kernel Linux 4.4 și manager de servicii systemd.
  • Este certificat ca oaspete pe AWS, Microsoft Azure, Joyent, IBM și HP Cloud.
  • Suport Netplan (disponibil de la Ubuntu 16.04 LTS)
  • manager de pachete snap
  • Suport de criptare LVM

    • Are versiuni actualizate ale software-ului serverului cheie preinstalat, inclusiv: Tomcat (v8), PostgreSQL (v9.5), Docker v (1.10), Puppet (v3.8.5), Qemu (v2.5), Libvirt (v1.3.1), LXC (v2.0) și MySQL (v5.6).
