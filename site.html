Ubuntu este un sistem de operare open-source (OS) bazat pe distribuția Debian GNU / Linux. Ubuntu încorporează toate caracteristicile unui sistem de operare Unix cu un GUI personalizabil adăugat, ceea ce îl face popular în universități și organizații de cercetare. Ubuntu este conceput în primul rând pentru a fi folosit pe computere personale, deși există și ediții de server. Ubuntu este un cuvânt african care înseamnă literal „umanitate pentru ceilalți”. Ubuntu a fost lansat pentru prima dată în 2004. Proiectul este sponsorizat de Canonical Ltd., o companie din U.K., care generează venituri prin vânzarea de asistență și servicii pentru a completa Ubuntu. Canonical lansează o nouă versiune a Ubuntu la fiecare șase luni și oferă suport sub formă de patch-uri și versiuni de securitate timp de 18 luni după aceea. Ubuntu este format din multe pachete software, care sunt autorizate sub licența publică generală GNU. Acest lucru permite utilizatorilor să copieze, să modifice, să dezvolte și să redistribuie propria versiune a programului. Ubuntu vine cu o gamă largă de programe software, inclusiv FireFox și LibreOffice. Există, de asemenea, un software proprietar care poate fi rulat pe Ubuntu.
